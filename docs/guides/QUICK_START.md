# Quick Start Guide

## Getting Started

### macOS

1. Open `EmailStampApp.xcodeproj` in Xcode
2. Select the **EmailStampApp (macOS)** scheme
3. Press ⌘R to build and run

### iOS

1. Open `EmailStampApp.xcodeproj` in Xcode
2. Select the **EmailStampApp (iOS)** scheme
3. Choose a simulator or connected device
4. Press ⌘R to build and run

## First Steps

1. **Compose an Email**: Click "New" to start composing
2. **Add Recipient**: Enter the recipient's email address
3. **Write Subject**: Add your email subject
4. **Write Body**: Compose your message
5. **Add Stamp** (optional): Click "Add Stamp" to include a visual stamp
6. **Configure PGP** (optional): Click "PGP Settings" to set up signing
7. **Send**: Click "Send" to generate your verified email

## Features

- ✅ Cryptographic verification with SHA256
- ✅ QR code generation
- ✅ Visual stamps
- ✅ PGP signing support
- ✅ Dark mode support
- ✅ Cross-platform (macOS, iOS, iPadOS)
